# User stories index

The user stories for the project have been separated into role-specific files under this folder:

- `user-stories/Student.md` — Student user stories
- `user-stories/Teacher.md` — Teacher user stories
- `user-stories/Parent.md` — Parent user stories
- `user-stories/Administrator.md` — Administrator / School Admin user stories

These files were extracted from `src/App.tsx` where the role content arrays were originally defined.
